1.linux目录之/和./和../以及~的区别
https://blog.csdn.net/Laurel1115/article/details/87975285
/：代表着根目录，是树形结构的最上层
.：表示当前目录，也可以用./表示；
. .：表示上一级目录，也可以用. ./表示；
~：代表用户自己的家目录；

2.linux中cd / cd~ cd cd- 和cd..之间的区别
https://www.cnblogs.com/torchstar/p/9976841.html
cd        进入用户主目录 
cd ~     进入用户主目录 
cd -      返回进入此目录之前所在目录 
cd ..     返回上一级目录 
cd ../..  返回上两级目录 
cd !$    把上个命令的参数作为cd 参数使用 
cd /      进入根目录
cd .      当前目录
